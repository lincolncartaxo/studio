# **App Name**: Greenlyfe Store

## Core Features:

- Product Catalog: Display products categorized into grains, supplements, and juices/shakes.
- Product Details: Show detailed information for each product, including description and price.
- WhatsApp Order: Redirect users to WhatsApp with a pre-filled message containing order details when the purchase is completed, sent to +558399999999.
- AI Nutrition Assistant: Provide personalized product recommendations based on user's dietary needs and preferences by acting as a tool for healthy lifestyle choices.

## Style Guidelines:

- Primary color: Fresh green (#77B355) to represent nature and health.
- Background color: Light beige (#F5F5DC), creating a natural and calming backdrop.
- Accent color: Earthy brown (#A0522D) for highlighting calls to action and important elements.
- Body and headline font: 'PT Sans', a humanist sans-serif that is modern, clean and readable.
- Use simple and clean icons to represent each product category (grains, supplements, juices/shakes).
- Responsive layout that adapts to different screen sizes for optimal viewing experience on all devices.
